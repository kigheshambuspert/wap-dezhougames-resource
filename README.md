# wap-dezhougames-resource

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对任何特定域名或具体网站，仅作为前端资源存档使用。

## 项目简介

这是一个资源存档仓库，主要存放静态 HTML 页面及相关前端资源文件。所有页面均为独立运行的结构，不依赖外部特定服务或数据库。

## 目录说明

```
/
├── pages/          # 存放各个独立 HTML 页面
├── assets/         # 共用资源文件（CSS、JavaScript、图片等）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立页面，页面之间互不依赖。
- `assets/`：存放页面可能引用的公共资源，如样式表、脚本、字体或图像。

## 页面归档说明

- 每个页面均为完整的 HTML 文件，可直接在浏览器中打开运行。
- 页面内可能引用 `assets/` 目录下的资源，引用路径为相对路径。
- 归档内容保持原始结构，不进行额外修改或适配。
- 如页面需要外部资源（如 CDN 库），会在页面头部注明来源。

## 维护说明

- 本仓库仅用于存档和发布，不提供实时更新或技术支持。
- 如需添加新页面，请将 HTML 文件放入 `pages/` 目录，并确保资源引用正确。
- 不建议对已有归档页面进行结构性修改，以免破坏原有功能。
- 如有问题或建议，可通过 Issues 提交，但不保证及时响应。
